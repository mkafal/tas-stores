# TAS Stores — Official Website

Modern website for TAS Stores, Sierra Leone's leading certified cocoa and coffee exporter.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **Fonts**: Playfair Display (headings) + Inter (body)
- **Language**: TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home — Hero, stats, certifications, supply chain, CTA |
| `/about` | Company story, timeline, process steps |
| `/products` | Cocoa & Coffee specs with full export terms |
| `/sustainability` | Impact metrics, traceability chain, SDG alignment |
| `/achievements` | Awards and government recognition |
| `/contact` | Structured RFQ form + WhatsApp CTA |
